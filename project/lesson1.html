Урок 1. Подходы к измерению информации

1. Что такое информация?

Слово «информация» происходит от латинского informatio (сведение, разъяснение). В широком смысле информация — это любые сведения об окружающем мире, которые человек получает через органы чувств. В информатике сложилось несколько подходов к измерению информации, каждый из которых удобен в своей области.

  • Традиционный (обыденный): информация — это знания, сообщения, новости. Этот подход не даёт числовой меры.
  • Вероятностный (содержательный): информация — это сведения, уменьшающие неопределённость знаний. Чем больше неопределённость снимается, тем больше информации.
  • Алфавитный (объёмный): информация рассматривается как последовательность символов, и её количество зависит от длины сообщения и мощности алфавита.

2. Бит — единица измерения информации

В компьютере вся информация представляется двоичным кодом (0 и 1). Один двоичный знак (0 или 1) называется битом (от англ. binary digit). Бит — наименьшая единица измерения количества информации.

С точки зрения содержательного подхода, сообщение, уменьшающее неопределённость ровно в два раза, несёт 1 бит информации.

3. Содержательный (вероятностный) подход. Формула Хартли

Представим, что мы ожидаем событие, которое может произойти с некоторой вероятностью. До получения сообщения у нас была неопределённость (несколько равновероятных вариантов). После получения сообщения неопределённость исчезает. Количество информации определяется тем, насколько уменьшилась неопределённость.

Если все N возможных событий равновероятны, то количество информации I, содержащееся в сообщении о том, что произошло одно из них, вычисляется по формуле Хартли:

N = 2I    или    I = log2 N

Здесь N — число равновероятных событий, I — количество информации в битах.

Примеры

Пример 1. Подбрасывание монеты: возможно два исхода (орёл или решка). N = 2. По формуле I = log22 = 1 бит. Сообщение о результате даёт 1 бит информации.

Пример 2. В коробке 8 разноцветных карандашей. Вы случайно вынимаете один. Сколько информации несёт сообщение о цвете карандаша?

N = 8 (все карандаши различны и равновероятны). I = log28 = 3 бита.

Пример 3. Игральный кубик имеет 6 граней. N = 6, I = log26 ≈ 2,585 бита. Количество информации может быть дробным.

4. Алфавитный подход

Этот подход используется в технике для определения объёма памяти, необходимого для хранения текста. Здесь не важна вероятность появления символов — каждый символ текста имеет определённый информационный вес.

Пусть алфавит некоторого языка содержит N символов. Тогда каждый символ можно закодировать с помощью i бит так, чтобы выполнялось соотношение:

N ≤ 2i ⇒ i = ⌈log2 N⌉ (округляем вверх)

Величина i называется информационным весом символа. Тогда информационный объём сообщения, состоящего из K символов, равен:

Iтекст = K · i (бит)

Примеры

Пример 4. Алфавит племени содержит 16 символов. Каков информационный вес одного символа?

N = 16, i = log216 = 4 бита.

Пример 5. Сообщение, записанное буквами из 32‑символьного алфавита, содержит 140 символов. Какое количество информации оно несёт?

i = log232 = 5 бит. I = 140 × 5 = 700 бит.

Пример 6. Сколько символов в сообщении, если его объём равен 1 Кбайт, а алфавит состоит из 256 символов?

1 Кбайт = 1024 байт = 8192 бит. i = log2256 = 8 бит = 1 байт. Значит, K = 8192 / 8 = 1024 символа.

5. Связь между подходами

Содержательный подход говорит о смысловой ценности информации, алфавитный — о техническом объёме. Например, одно и то же сообщение на разных языках (разные алфавиты) будет иметь разный объём, хотя смысл одинаков. В компьютерах всегда используется алфавитный подход: любой файл имеет размер в байтах, независимо от содержания.
