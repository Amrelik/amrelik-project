Урок 6. Основы алгебры логики

1. Логические высказывания и переменные

Высказывание — это повествовательное предложение, о котором можно сказать, истинно оно или ложно. Например: «Земля вращается вокруг Солнца» — истина; «5 > 10» — ложь. Вопросительные и восклицательные предложения высказываниями не являются.

В алгебре логики высказывания обозначают логическими переменными (обычно латинскими буквами A, B, C…). Каждая переменная может принимать одно из двух значений: истина (1) или ложь (0).

2. Логические операции

Над высказываниями можно выполнять операции, результатом которых также будет истина или ложь. Рассмотрим основные операции.

2.1. Инверсия (отрицание, НЕ, ¬, черта сверху)

Инверсия меняет значение высказывания на противоположное. Обозначается: ¬A или A̅.

A¬A
01
10

Пример: A = «Сегодня понедельник». ¬A = «Сегодня не понедельник».

2.2. Конъюнкция (логическое И, AND, &, ∧)

Конъюнкция двух высказываний истинна только тогда, когда оба высказывания истинны. Обозначается: A ∧ B, A & B, A and B.

ABA ∧ B
000
010
100
111

Пример: A = «Число чётное», B = «Число больше 10». A ∧ B означает «Число чётное и больше 10» (например, 12, 14…).

2.3. Дизъюнкция (логическое ИЛИ, OR, ∨)

Дизъюнкция двух высказываний ложна только тогда, когда оба высказывания ложны. Истинна, если хотя бы одно истинно. Обозначается: A ∨ B, A or B.

ABA ∨ B
000
011
101
111

Пример: A = «Студент сдал экзамен», B = «Студент получил зачёт». A ∨ B означает «Студент сдал экзамен или получил зачёт» (возможно и то, и другое).

2.4. Импликация (логическое следование, →)

Импликация A → B ложна только в одном случае: когда A истинно, а B ложно. Во всех остальных случаях истинна. A называют посылкой, B — следствием. Обозначается: A → B.

ABA → B
001
011
100
111

Пример: A = «Число делится на 4», B = «Число чётное». Тогда A → B означает: «Если число делится на 4, то оно чётное» — это истинное высказывание.

2.5. Эквиваленция (логическое равенство, ↔, ≡)

Эквиваленция истинна тогда, когда оба высказывания одновременно истинны или одновременно ложны. Обозначается: A ↔ B, A ≡ B.

ABA ↔ B
001
010
100

Пример: A = «Треугольник равносторонний», B = «Все углы треугольника равны 60°». A ↔ B истинно.

3. Приоритет логических операций

Если в выражении нет скобок, сначала выполняется отрицание (НЕ), затем конъюнкция (И), затем дизъюнкция (ИЛИ), затем импликация и эквиваленция (обычно они имеют самый низкий приоритет). Для изменения порядка используются скобки.

4. Таблицы истинности сложных выражений

Для любого логического выражения можно построить таблицу истинности, перебирая все возможные комбинации значений переменных и вычисляя результат. Количество строк в таблице равно 2n, где n — число переменных.

5. Основные законы алгебры логики

Законы позволяют упрощать логические выражения и преобразовывать их к нужному виду. Ниже приведены наиболее важные тождества (здесь 1 — истина, 0 — ложь).

  • Закон тождества: A = A
  • Закон непротиворечия: A ∧ ¬A = 0
  • Закон исключённого третьего: A ∨ ¬A = 1
  • Закон двойного отрицания: ¬¬A = A
  • Идемпотентность: A ∧ A = A, A ∨ A = A
  • Коммутативность: A ∧ B = B ∧ A, A ∨ B = B ∨ A
  • Ассоциативность: (A ∧ B) ∧ C = A ∧ (B ∧ C), (A ∨ B) ∨ C = A ∨ (B ∨ C)
  • Дистрибутивность: A ∧ (B ∨ C) = (A ∧ B) ∨ (A ∧ C), A ∨ (B ∧ C) = (A ∨ B) ∧ (A ∨ C)
  • Законы поглощения: A ∧ (A ∨ B) = A, A ∨ (A ∧ B) = A
  • Законы де Моргана:
    ¬(A ∧ B) = ¬A ∨ ¬B
    ¬(A ∨ B) = ¬A ∧ ¬B
  • Свойства констант: A ∧ 1 = A, A ∧ 0 = 0, A ∨ 1 = 1, A ∨ 0 = A

6. Примеры упрощения логических выражений

Используя законы, можно упрощать сложные выражения. Рассмотрим несколько примеров.

Пример 1: Упростить (A ∧ B) ∨ (A ∧ ¬B).

Решение: Применим закон дистрибутивности: A ∧ (B ∨ ¬B) = A ∧ 1 = A. Ответ: A.

Пример 2: Упростить ¬(A ∨ B) ∨ (¬A ∧ B).

Решение: По закону де Моргана ¬(A ∨ B) = ¬A ∧ ¬B. Тогда выражение принимает вид (¬A ∧ ¬B) ∨ (¬A ∧ B) = ¬A ∧ (¬B ∨ B) = ¬A ∧ 1 = ¬A. Ответ: ¬A.

Пример 3: Упростить (A → B) ∧ A.

Решение: Заменим импликацию: A → B = ¬A ∨ B. Тогда (¬A ∨ B) ∧ A = (¬A ∧ A) ∨ (B ∧ A) = 0 ∨ (A ∧ B) = A ∧ B. Ответ: A ∧ B.

7. Логические функции и способы их задания

Логическая функция — это выражение, зависящее от логических переменных. Её можно задать:

  • аналитически (формулой);
  • таблицей истинности;
  • словесным описанием;
  • логической схемой.

8. Применение алгебры логики

8.1. Логические схемы компьютера

Элементарные логические операции реализуются электронными элементами — вентилями. Конъюнкция — вентиль И, дизъюнкция — вентиль ИЛИ, отрицание — вентиль НЕ. Комбинируя их, строят более сложные узлы процессора: сумматоры, триггеры, регистры.

8.2. Поисковые запросы

В поисковых системах логические операции используются для уточнения запроса. Например, запрос информатика & ЕГЭ означает, что страница должна содержать оба слова (конъюнкция). Запрос информатика | ЕГЭ — любое из слов (дизъюнкция). Оператор ! или - часто обозначает отрицание.

Пример. Пусть A — множество страниц, содержащих слово «кошки», B — слово «собаки». Тогда запрос A & B выдаст страницы, где есть оба слова, A | B — где есть хотя бы одно, A & ¬B — где есть «кошки», но нет «собаки».
